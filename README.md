# FatturaElettronicaToPDF

Print one or more Electronic Invoices into a single PDF.


# Installation

Install wkhtmltopdf from https://wkhtmltopdf.org/downloads.html.

Add pdfkit library to Python3: `pip3 install pdfkit`.
